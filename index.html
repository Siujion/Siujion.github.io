PYTHON PROGRAMMING - My experience as a beginner

As part of my MSC core modules at the University of Leeds, I learned the basics of programming using Python. Learning programming has been very challenging given that I have no background in Information Systems or Programming. However, I have come to understand some of the basics where I've learned how to write, debug and run a code for Agent Based Modelling. The first model that I created as part of this course is an agent based modelling. The codes were provided to me as part of my practical, in which I had to make adjustments, run and debug it.The code specifically creates an environment using pixel values of raster data, it also creates agents, their behaviour (move and eat) and a graphic user interface. The agents in this model represent the invasive Crazy yellow ants which are capable of destroying vast ecosystems. In this model, I animate how the ants inhabit and destroy an ecosystem and how an increase in temperature within 100 seconds can wipe out their population.
